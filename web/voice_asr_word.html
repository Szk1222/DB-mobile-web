Overview · 产品介绍

什么是一句话识别

一句话识别（Short Speech Recognition）是标贝智能语音识别（ASR）产品线中面向短音频的实时转写服务，专为 60 秒以内的语音输入设计。用户边说话，引擎边以流式分片返回识别文本，真正实现"随说随出字"的交互体验。

产品依托标贝自研的端到端深度神经网络声学模型，结合大规模真实场景语料持续训练迭代。在标准测试集上普通话识别准确率达 98% 以上，并内置前端降噪、智能断句、标点预测与数字格式规整能力，输出可直接使用的规范文本。

无论是语音输入法、智能硬件语音指令，还是 App 内的语音交互入口，一句话识别都能以低于 300ms 的首字延迟稳定支撑。产品同时提供公有云 API 与私有化部署两种接入方式，满足开发者快速验证与企业级合规落地的不同需求。

Core Features · 核心功能

六大能力开箱即用

实时流式识别

音频分片上传，引擎边接收边解码，识别结果随说随返，端到端首字延迟低于 300ms，适配实时语音交互。

智能断句与标点

基于语义模型自动添加逗号、句号、问号等标点，自动断句分行，输出符合阅读习惯的规范文本。

数字与格式规整

内置 ITN 逆文本规整，将"一百二十三"转为"123"，自动识别金额、日期、时间、电话号码等结构化文本。

多语种与方言

支持普通话、英语、粤语、四川话、上海话等 20+ 语种方言，并可识别中英文混读内容。

热词与领域定制

上传专有名词、品牌词、人名地名等热词表，针对性提升垂直领域专有词汇的识别召回率。

噪声抑制与回声消除

前端集成信号处理算法，有效抑制背景噪声与回声，适配车载、户外、公共空间等强噪环境。

API Reference · 接口说明

快速对接接口

请求核心参数与多语言调用示例，30 分钟即可完成首次接入

参数类型必填说明
app_idstring必填应用 ID，在开放平台创建应用后获取
api_keystring必填接口鉴权密钥，请妥善保管，勿在前端明文暴露
audio_formatstring必填音频编码：pcm / wav / mp3 / opus / speex
sample_rateint必填采样率，取值 8000 或 16000
languagestring选填识别语种，默认 zh-CN，支持 en-US、yue-CN 等
enable_punctuationbool选填是否添加标点符号，默认 true
enable_itnbool选填数字 / 金额 / 日期格式规整，默认 true
hotword_idstring选填热词表 ID，用于提升专有名词识别率
 
import websocket, json

URL = "wss://openspeech.data-baker.com/asr/v1/short"
ws = websocket.create_connection(
    URL, header=["Authorization: your_api_key"])

# 1. 发送识别参数
ws.send(json.dumps({
    "app_id": "your_app_id",
    "audio_format": "pcm",
    "sample_rate": 16000,
    "language": "zh-CN",
    "enable_punctuation": True,
    "enable_itn": True
}))

# 2. 分片发送音频
with open("audio.pcm", "rb") as f:
    for chunk in iter(lambda: f.read(3200), b""):
        ws.send_binary(chunk)
ws.send_binary(b"")          # 发送结束标记

# 3. 接收识别结果
print(json.loads(ws.recv())["result"]["text"])

Scenarios · 应用场景

短语音识别典型落地

替代键盘输入，用户口述即可转写为文字，输入效率提升数倍，覆盖聊天、搜索、备忘等场景。

智能音箱、机器人、可穿戴设备的指令识别，准确捕捉唤醒后的简短命令并即时响应。

电商、地图、视频等 App 内语音搜索入口，口述关键词即时转写并触发检索。

App 内语音交互

为移动应用接入语音助手、语音指令等交互入口，丰富无障碍与免手操作体验。
