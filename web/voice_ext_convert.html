Overview · 产品介绍

关于声音转换

声音转换（Voice Conversion）能够将一段语音中说话人的音色替换为目标说话人，同时完整保留原始的语义内容、语调节奏与情感表达。简单来说，“说什么、怎么说”不变，只改变“是谁的声音”。

区别于简单的变调变声，标贝声音转换基于深度神经网络建模音色与内容的解耦表示，转换后的语音自然流畅、无机械感，目标音色还原度高，并支持实时流式处理。

产品广泛应用于影视配音、游戏与虚拟形象、直播娱乐互动、声音隐私保护等场景。功能须在合法合规、获得相关方授权的前提下使用，平台提供授权校验与内容追溯机制。

核心请求参数与调用示例，便于快速完成接入

参数类型必填说明
app_idstring必填应用 ID，开放平台创建应用后获取
api_keystring必填接口鉴权密钥
target_voicestring必填目标音色标识
audio_formatstring必填音频格式：pcm / wav / mp3

import websocket, json

URL = "wss://openspeech.data-baker.com/vc/v1/convert"
ws = websocket.create_connection(
    URL, header=["Authorization: your_api_key"])

ws.send(json.dumps({
    "app_id": "your_app_id",
    "target_voice": "magnetic_male",
    "audio_format": "pcm",
    "sample_rate": 16000,
    "mode": "stream"
}))

with open("source.pcm", "rb") as f:
    for chunk in iter(lambda: f.read(3200), b""):
        ws.send_binary(chunk)
ws.send_binary(b"")

# 接收转换后的音频流
with open("converted.pcm", "wb") as out:
    while True:
        data = ws.recv()
        if not data:
            break
        out.write(data)
