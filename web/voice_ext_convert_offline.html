Overview · 产品介绍

关于离线声音转换

离线声音转换（Offline Voice Conversion）将声音转换能力完整部署到设备端。转换模型在本地运行，无需联网即可实时将说话人音色替换为目标音色，并保留原始的语义内容与说话风格。

引擎针对端侧算力做了深度优化，体积小、延迟低、功耗可控，可在移动设备、嵌入式终端等多种硬件上稳定运行。音频数据全程不出设备，从根本上保障声音隐私安全。

离线声音转换适用于对实时性、数据合规要求高的场景，如本地化的变声互动、隐私敏感的语音处理、无网络环境下的内容创作等。

API Reference · 接口说明

快速对接接口

核心请求参数与调用示例，便于快速完成接入

参数类型必填说明
license_pathstring必填设备授权文件路径
model_pathstring必填离线转换模型资源包路径
target_voicestring必填目标音色标识
sample_rateint选填采样率：16000 / 24000
modestring选填转换模式：stream / batch，默认 stream
 
// 初始化离线声音转换引擎
OfflineVC vc = new OfflineVC();
vc.init(getAssets(), "license.dat", "model_vc");
vc.setTargetVoice("offline_male_a");

// 实时转换音频流
vc.start(new VCCallback() {
    @Override public void onConverted(byte[] pcm) {
        audioTrack.write(pcm, 0, pcm.length);
    }
});
vc.feed(micPcmData);   // 持续送入麦克风音频
vc.stop();
vc.release();
