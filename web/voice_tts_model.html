Overview · 产品介绍

关于大模型声音复刻

大模型声音复刻（LLM Voice Cloning）依托标贝自研的语音大模型，将声音复刻带入“秒级、零样本”时代。仅需一段数秒的录音，模型即可捕捉说话人的音色特征、发音习惯与韵律风格，生成高度拟真的专属音色。

区别于传统定制方案，大模型复刻无需大规模录音棚采集，也无需长时间训练。模型具备强大的泛化与情感表现能力，复刻音色可自然朗读任意文本，并支持高兴、平静、严肃等多种情感与语速、语调的细粒度控制。

产品适用于有声内容创作、个性化语音助手、数字人配音等场景。为保障声音权益与合规使用，复刻功能需在获得本人明确授权的前提下使用，平台提供完整的授权校验与水印追溯机制。

API Reference · 接口说明

快速对接接口

核心请求参数与调用示例，便于快速完成接入

参数类型必填说明
app_idstring必填应用 ID，开放平台创建应用后获取
api_keystring必填接口鉴权密钥
voice_idstring必填复刻音色 ID，复刻完成后生成
textstring必填待合成文本，单次 ≤ 1000 字
emotionstring选填情感风格：neutral / happy / serious 等
speedfloat选填语速，范围 0.5 ~ 2.0，默认 1.0
audio_formatstring选填输出格式：wav / mp3 / pcm，默认 wav
