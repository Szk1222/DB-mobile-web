Overview · 产品介绍

关于离线合成

离线合成（Offline TTS）是标贝面向端侧设备的本地语音合成引擎。合成模型完整运行在设备本地，无需联网即可将文字实时转换为自然语音，彻底摆脱网络延迟与连接不稳定的影响。

引擎针对嵌入式平台深度优化，体积小、内存占用低、功耗可控，可在 ARM、DSP、国产芯片等多种硬件上流畅运行。即使在弱网或完全无网络的环境下，也能保障语音交互的即时响应。

由于音频数据无需上传云端，离线合成天然具备隐私安全优势，特别适合车载终端、智能家居、可穿戴设备、工业设备等对数据合规、响应实时性要求高的场景。

API Reference · 接口说明

快速对接接口

核心请求参数与调用示例，便于快速完成接入

参数类型必填说明
license_pathstring必填设备授权文件路径
model_pathstring必填离线模型资源包路径
voicestring必填发音人标识
textstring必填待合成文本
speedfloat选填语速，范围 0.5 ~ 2.0，默认 1.0
sample_rateint选填采样率：16000 / 24000
 
// 初始化离线合成引擎
OfflineTTS tts = new OfflineTTS();
tts.init(getAssets(), "license.dat", "model_zhiknv");

// 合成并播放
tts.setSpeed(1.0f);
tts.setVoice("standard_female");
tts.speak("离线语音合成，无需联网。", new TTSCallback() {
    @Override public void onAudio(byte[] pcm) {
        audioTrack.write(pcm, 0, pcm.length);
    }
    @Override public void onFinish() { /* 合成完成 */ }
});

// 释放资源
tts.release();
