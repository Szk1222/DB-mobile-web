Overview · 产品介绍

关于在线合成

在线合成（Online TTS）是标贝语音合成产品线的核心在线服务，将任意文本实时转换为高音质语音。引擎采用端到端神经网络合成方案，音色自然饱满，韵律贴近真人发音。

产品提供数百款发音人音色，覆盖通用、客服、播报、儿童、方言等多种风格，并支持高兴、平静等情感表达与语速、语调、音量的细粒度控制。在线合成首包延迟低于 150ms，支持流式返回，可边合成边播放。

通过 SSML 标记语言，开发者还可精确控制多音字读音、数字读法、停顿与重音。在线合成广泛用于智能客服、有声阅读、新闻播报、语音通知等需要实时、高质量语音输出的场景。

API Reference · 接口说明

快速对接接口

核心请求参数与调用示例，便于快速完成接入

参数类型必填说明
app_idstring必填应用 ID，开放平台创建应用后获取
api_keystring必填接口鉴权密钥
textstring必填待合成文本或 SSML，单次 ≤ 1000 字
voicestring必填发音人标识，如 zhiknv / wennan
audio_formatstring选填输出格式：wav / mp3 / pcm / opus，默认 wav
sample_rateint选填采样率：16000 / 24000 / 48000
speedfloat选填语速，范围 0.5 ~ 2.0，默认 1.0
pitchfloat选填语调，范围 0.5 ~ 2.0，默认 1.0
volumefloat选填音量，范围 0.1 ~ 2.0，默认 1.0
 
import requests

resp = requests.post(
    "https://openspeech.data-baker.com/tts/v1/online",
    headers={"Authorization": "your_api_key"},
    json={
        "app_id": "your_app_id",
        "text": "标贝在线语音合成，音色自然饱满。",
        "voice": "zhiknv",
        "audio_format": "mp3",
        "sample_rate": 24000,
        "speed": 1.0
    })

with open("output.mp3", "wb") as f:
    f.write(resp.content)
curl -X POST "https://openspeech.data-baker.com/tts/v1/online" \
  -H "Authorization: your_api_key" \
  -H "Content-Type: application/json" \
  -o output.mp3 \
  -d '{
        "app_id": "your_app_id",

        "voice": "zhiknv",

      }'
