Overview · 产品介绍

关于声音复刻

声音复刻（Voice Cloning）用少量用户录音即可定制一款专属个人音色。相比传统需要数千句录音棚采集的音库定制方案，声音复刻将所需录音量与制作周期大幅压缩，让个性化音色“人人可得”。

用户按提示录制一定数量的引导语句，系统即可学习其音色特征并完成建模。复刻音色可朗读任意文本，并支持情感与风格控制，在不同场景下都能保持稳定一致的声音表现。

声音复刻在成本、周期与效果之间取得良好平衡，适合需要批量定制音色的内容平台、教育机构与企业客户。功能须在获得声音本人授权的前提下使用，平台提供完整授权与合规保障。

API Reference · 接口说明

快速对接接口

核心请求参数与调用示例，便于快速完成接入

参数类型必填说明
app_idstring必填应用 ID，开放平台创建应用后获取
api_keystring必填接口鉴权密钥
voice_idstring必填复刻音色 ID，复刻完成后生成
textstring必填待合成文本，单次 ≤ 1000 字
emotionstring选填情感风格：neutral / happy / sad 等
speedfloat选填语速，范围 0.5 ~ 2.0，默认 1.0
 
import requests

resp = requests.post(
    "https://openspeech.data-baker.com/tts/v1/reprint",
    headers={"Authorization": "your_api_key"},
    json={
        "app_id": "your_app_id",
        "voice_id": "reprint_2c7f9a",
        "text": "这是用我的定制音色合成的语音。",
        "emotion": "neutral",
        "speed": 1.0
    })

with open("output.wav", "wb") as f:
    f.write(resp.content)
